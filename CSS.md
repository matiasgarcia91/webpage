# CSS => Cascading Style Sheet

```css
// Selector (.color_red)
.color-red {
  color: red  // (property: value)
  border: 3px dotted black;
}
```

## Selectors

1. Tag type (element selector)

```css
h2 {
}
```

2. Class selector or class name

```css
.vertical-spacing {
  margintop: 10px;
  marginbottom: 10px;
}
```

3. By Id

```css
#text-description {
  color: blue;
}
```

4. Pseudo-class selectors:

```css
a:hover {
}
```
